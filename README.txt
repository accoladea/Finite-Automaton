# Finite Automaton Pattern Matcher

* The program matches specific defined patterns and prints true if the defined patterns are matching

* The project is done by collaboration of Islomzhan Akhmedov and Abdullah Alsayari

* On project folder from terminal type, “make” to build and “./main” to check it

* To test different inputs find on main.c “nfa_inputs” and “dfa_inputs” and test them with different inputs
